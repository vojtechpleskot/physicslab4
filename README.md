# Physics Lab 4

English version of Jupyter notebooks for the Physics Lab 4 course at the Charles University in Prague.
This is a work in progress, so expect some errors and missing content. If you find any, please report it to me.
The Czech version of the notebooks is here: https://github.com/physicslab4cuni-dot/physicslab4.
The Czech version is the original one, and it is used in the course.
The English version is a translation of a somewhat outdated version of the Czech notebooks, so it may contain some errors and missing content.
The differences should not be critical, but be aware of the existence of the better, clearer, and more maintained Czech version.

# Test data

Some notebooks are empty and waiting for the data and code snippets to be inserted.
In the notebooks, places where data and code snippets are missing are marked with "TODO" comments.
An example data and solution of the missing content is available in the `test_data` folder.
The exact text to be inserted to the notebooks is located in the `test_data/A<X>/code.txt` file.
Input data and example image files are typically already copied from the `test_data/A<X>` folder to the corresponding `A<X>` folder containing the Jupyter notebook.
Let me know if something is missing in the test data!
